### HTML 설정
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script defer src="variable.js"></script>
    <!-- defer : js를 다운받으라고 명령만 해놓고 html 파일을 끝까지 parsing(해석)하는 동안
        js파일을 다운받은 뒤 html parsing(해석)이 끝나면 js를 실행한다 -->
      
</head>
<body>
    
</body>
</html>

```
